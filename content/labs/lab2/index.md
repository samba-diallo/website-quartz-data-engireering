## Description

Ce lab approfondit les concepts de data warehousing avec Spark, notamment :

- Architecture des data warehouses
- Spark SQL et optimisation des requetes
- Partitioning et bucketing
- Schema design et optimisation
- Perf de performance pour les warehouses de donnees
